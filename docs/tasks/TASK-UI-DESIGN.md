# 文心课堂 — UI/UX 设计规范任务书

> 派发给：UI 设计 Agent | 日期：2026-05-10

---

## 任务目标

为"文心课堂"在线教育平台制定完整的 UI/UX 设计规范，包括视觉风格、配色方案、组件规范、页面布局线框图。

---

## 项目背景

- **项目类型：** 在线教育平台（品牌宣传 + 课程售卖 + 视频学习）
- **目标用户：** 国内 18-45 岁，想学习新技能的职场人和学生
- **设备适配：** PC 端 + 移动端（响应式）
- **技术栈：** Next.js + Tailwind CSS + shadcn/ui

---

## 任务要求

### 1. 设计风格定位
分析 2-3 种适合在线教育的视觉风格，推荐一种并说明理由：
- 现代简约风（类似 Notion、Linear）
- 活力教育风（类似 Duolingo、猿辅导）
- 专业商务风（类似 Coursera、极客时间）

### 2. 配色方案
提供完整的配色方案，包含：
- 主色（品牌色）1-2 个
- 辅助色
- 背景色（亮色模式）
- 文字色（标题/正文/辅助）
- 状态色（成功/警告/错误/信息）
- 暗色模式配色（可选）

### 3. 字体方案
- 中文字体推荐（系统字体栈）
- 英文/数字字体
- 字号层级（h1-h6, body, caption）
- 行高、字间距规范

### 4. 组件规范
基于 shadcn/ui，定义以下组件的样式规范：
- 按钮（主要/次要/危险/禁用状态）
- 卡片（课程卡片、讲师卡片）
- 表单（输入框、选择器、搜索框）
- 导航（顶部导航、移动端汉堡菜单、底部导航）
- 弹窗/抽屉
- 标签/徽章
- 加载状态

### 5. 页面布局线框图
用 ASCII/文字描述以下页面的布局结构（不需要视觉稿，只需要布局框架）：

**前台页面：**
- 首页（PC + Mobile）
- 课程列表页
- 课程详情页
- 学习页面（视频播放）
- 个人中心

**后台页面：**
- Dashboard
- 课程管理列表
- 课程编辑表单
- 用户管理

### 6. 响应式断点
定义断点规范：
- Mobile: < 768px
- Tablet: 768px - 1024px
- Desktop: > 1024px
- 各断点下的布局变化规则

### 7. 动效规范
- 页面切换动效
- 卡片 hover 效果
- 按钮交互反馈
- 加载动画
- 滚动动画（可选）
